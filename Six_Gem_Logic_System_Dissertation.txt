ABSTRACT
======================================================================
Traditional Boolean logic is binary (True/False) and operates along a one-dimensional line.
6 Gem Logic is a ternary stream inference system built on a hexagonal phase manifold.
It treats truth not as a static boolean value, but as a phase position within a continuous cyclic stream generated by the irreducible interaction of three elements.
The core logical operator is the Stream Inference [A, B, C], where two premises (A, B) are brought into interaction and “collapsed” into a definite phase state by the third element (the Stream-Witness C). 
The result is weakly irreducible: it cannot be fully reduced to successive binary operations without losing the joint triple configuration and orientation information.
All computational demonstrations are implemented in the companion Python system: Six_Gem_Logic_System_Dissertation_Suite.py.

======================================================================
I. THE 6 LOGICAL GEM STATES
======================================================================
The system replaces the classical True/False dichotomy with six discrete phase-states on a hexagonal cycle:

L0 (0): Absolute Affirmation (Identity / “It Is”)
L1 (1): Potential Flux (The “Could-Be” Echo)
L2 (2): Resonant Dissonance (The “Should-Be” Tension)
L3 (3): Absolute Negation (Inversion / “It Is Not”)
L4 (4): Reflective Flux (The “Could-Not-Be” Echo)
L5 (5): Resolved Dissonance (The “Should-Not-Be” Tension)

These states correspond to the 6 Gem rotational positions of the original Stereo-Identity algebra, and are color-coded in the system output for visual inspection.

======================================================================
II. THE STREAM INFERENCE OPERATOR [A, B, C]
======================================================================
Logic is performed via a single ternary operator called the Stream Inference:

[A, B, C] = (A + B + C + f(A,B,C) + χ(A,B,C)) mod 6

where:

f(A,B,C) = 1 if A, B, C are all distinct (high-resonance nonlinear correction), else 0
χ(A,B,C) = orientation (chirality) term: +1 for consistent short positive (clockwise) steps, –1 for consistent short negative (counter-clockwise) steps, 0 otherwise

This operator is weakly irreducible: the outcome depends on the joint triple configuration and the directional flow of the argument, not merely on pairwise combinations.

Example (from Six_Gem_Logic_System_Dissertation_Suite.py):

  Premises: A=L1, B=L3, Witness C=L2
  Stream Inference: [1, 3, 2] → L5 (Resolved Dissonance)
  Nonlinear Correction f(A,B,C)=1, Orientation χ(A,B,C)=0

======================================================================
III. CHIRALITY AND ORIENTATION IN LOGIC
======================================================================
A logical argument has an inherent handedness.

Cyclic flow (A → B → C with consistent short positive steps) adds +1 to the phase.
Anti-cyclic flow subtracts 1.
Mixed or stationary steps contribute 0.

Reversing the order of premises can produce a different collapsed state, introducing genuine non-commutativity and orientation sensitivity. 
This is quantitatively demonstrated in the system using the `order_sensitivity_test()` function:

  Example Output:
    Order sensitivity (A,B swap): 1101/2000 (≈55%)

======================================================================
IV. THE OBSERVER / STREAM-WITNESS EFFECT
======================================================================
In 6 Gem Logic there is no well-defined relationship between two premises alone.
Only when the third element (the Stream-Witness) enters does the stream “collapse” into one of the six definite gem states.
This mirrors the original ternary algebra’s non-associativity and context-dependence, and is computationally verified with the `associativity_test()` function:

  Example Output:
    Non-associativity rate: 1578/2000 (≈78.9%)

======================================================================

======================================================================
Weak ternary irreducibility
Non-associativity under composition
Orientation-sensitive (chiral) inference
Context-dependent nonlinear corrections
Remains close to the underlying Z6 cyclic structure while adding genuine triple-wise behavior

  - Deviation from Z6 baseline: `deviation_rate()`

The Python suite provides a full interactive demonstration of these properties, including random streams, manual input, trace mode, and visual phase-wheel representation.
